Projects

.NET Revenue Forecast

This is a lightweight, client-side web application built with ASP.NET Core 3, ML.NET, and Chart.js. The application reads in five years of historical revenue data and forecasts the following two years for financial reporting. ML.NET is using a fasttree regression machine learning model, chart.js renders a historical revenue graph for accuracy comparison, and Visual Studio 2019 was used to create a small bundle executable for the application.

Python Lotto Analysis

This is web scraping, data persistence, data retrieval API, and data visualization application. Pandas libraries are used to scrape the Colorado Lottery results website. The website data tables are converted to dataframes and persisted to an Azure database. An API is created to retrieve the .json file from the database and visualizations detailing the optimal lottory months are published to a Jupyter notebook using Matplotlib and Numpy.

.NET Form Salary Calculator

This is application is a simple Windows Form salary calculator using the .net framework. The form calculates lifetime wage to the age of sixty-five beginning at a user designated age, starting salary, and assuming a 5 percent annual increase for the remainder. Custom icons and background image were added to the form for aesthetic effect and empty fields generate errors messages requiring information.
